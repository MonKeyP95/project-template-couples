# LifeTogether

A beautiful app to help couples organize their shared life — tasks, calendar, budget, notes and more.

Built to be calm, collaborative, and easy to use. Future expansion to small teams planned.

## Tech Stack
- Next.js 15 + Tailwind CSS
- Supabase (Database + Auth)
- Vercel (Deployment)

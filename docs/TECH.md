# TECH.md

## Current Technology Stack

- **Frontend**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Deployment**: Vercel
- **Language**: TypeScript

## Why This Stack?

- Beginner friendly and easy to learn
- Excellent compatibility with Claude Code
- Scalable from a couples app → teams app later
- Great free tier for development
- Fast development speed
- Good documentation and community support

## Future Considerations
- Possible React Native for mobile app
- External API integrations (Google Calendar, payment systems, etc.)
- Shadcn/ui for beautiful components (can be added later)

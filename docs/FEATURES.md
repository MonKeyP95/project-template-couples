# FEATURES.md

## MVP (Must Have - Couples Version)
- User signup & login (one-time login)
- Couple pairing/invitation system (one partner invites the other)
- Shared Tasks (create, assign, complete, due dates)
- Shared Calendar (important dates, events)
- Shared Budget / Expense tracker
- Quick Notes & Lists
- Collection of data for analysis
- Agenda creation
- API integrations (Google Calendar, etc.)
- API integrations (Searching the web / Booking restaurants, ets.)
- AI chatbot

## Nice to Have
- Habit tracker for couples
- Reminders & notifications (no push notifications)
- Dark mode
- Anniversary & birthday reminders

## Future - Teams Version
- Multiple workspaces
- User roles & permissions
- Team task management
- Analytics & insights

## Current Phase
**Phase 1**: Project Setup & Foundation

## Development Approach
- Build iteratively — one small, working feature at a time
- Always keep the full vision in mind (Couples first → Teams later)
- Keep AI features modular so we can easily switch between OpenAI and Claude
- Focus on clean architecture from the beginning because of multiple API integrations and AI
- Test features with real couple usage in mind (simple, calm, collaborative)
- Document decisions regularly in `DECISIONS.md`

## Priorities
1. Make the core couple experience excellent first
2. Add powerful AI features after basic functionality works
3. Build flexible structure that can grow into Teams version

## Timeline Philosophy
We will move at a steady pace. Quality and learning > rushing features.
